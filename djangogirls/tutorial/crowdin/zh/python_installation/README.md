# 讓我們從 Python 開始吧

我們終於到了這裡！

但是首先，讓我們告訴你什麼是 Python。Python 是一種非常流行的程式設計語言，可以用來創建網站、 遊戲、 科學軟體、 圖形和很多，更多。

Python 起源于 1980 年代後期，其主要目標是針對人類的高可讀性 (不只是機器！)。 這就是為什麼它看起來比其他程式設計語言簡單得多。 這使得它容易學習，但不要擔心，Python 同時真的很強大！
